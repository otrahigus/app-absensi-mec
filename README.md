# 📸 Sistem Absen Wajah Anak Kampung

Sistem absensi otomatis berbasis pengenalan wajah yang bisa diakses dari 
HP.

## ✨ Fitur
- 📷 Absen dengan foto wajah langsung dari HP
- 📤 Upload foto wajah
- 📝 Daftar wajah baru
- 📊 Lihat rekap absensi
- ☁️ Data tersimpan otomatis ke Google Sheets

## 🚀 Cara Pakai
1. Buka link aplikasi
2. Daftar wajah di menu "Daftar Wajah Baru"
3. Absen di menu "Absen Wajah"
4. Lihat rekap di menu "Lihat Rekap"

## 📱 Akses
Bisa diakses dari HP maupun PC melalui browser.

## 🔧 Teknologi
- Streamlit
- DeepFace (face recognition)
- Google Sheets API
- OpenCV
